# Single reference
## Bistnet

Paper: <{{ "https://github.com/yyang181/NTIRE23-VIDEO-COLORIZATION" | uri_escape  }}>{:target="_blank"}

This method is originally designed for video colorization and is limited to 2 reference images.
The model is changed so that the input is 2 identical frames. The flow map is set to zeros and the similarity fusion is done using argmax for an arbitrary number of reference images.
